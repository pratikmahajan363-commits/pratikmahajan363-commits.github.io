# Pratik Mahajan Portfolio

This is a GitHub Pages portfolio website.

## How to publish
1. Create repo: `pratikmahajan363-commits.github.io`
2. Upload all files from this folder.
3. Go to Settings > Pages.
4. Select `Deploy from a branch`, branch `main`, folder `/root`.
5. Save and wait 1-2 minutes.

## Update before publishing
- Replace `your-email@example.com`
- Replace LinkedIn URL
- Add resume PDF named `Pratik_Mahajan_Resume.pdf`
- Add real project GitHub links if available
