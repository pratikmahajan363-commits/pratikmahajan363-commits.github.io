# Pratik Mahajan Portfolio

Personal BI/Data Analyst portfolio built with HTML, CSS, and GitHub Pages.

## How to publish

1. Create a GitHub repository named `pratikmahajan363-commits.github.io`
2. Upload `index.html`, `style.css`, and your resume PDF
3. Rename your resume to `Pratik_Mahajan_Resume.pdf`
4. Go to Settings → Pages
5. Select Branch: main and Folder: /root
6. Visit: https://pratikmahajan363-commits.github.io
